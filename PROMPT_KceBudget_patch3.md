## โครงสร้าง WBS — ต้องเข้าใจก่อน

```
C  - 68 - E - KCECS . 0080 . 02 . 2
│    │    │   │        │      │    └─ เจ้าของทรัพย์สิน: 1=กฟภ. / 2=ผู้ใช้ไฟ
│    │    │   │        │      └─── งบของ: 01=กฟภ. / 02=ผู้ใช้ไฟ / 03=อื่นๆ
│    │    │   │        └────────── หมายเลขแฟ้มงานหลัก
│    │    │   └─────────────────── การไฟฟ้า + แผนก
│    │    └─────────────────────── เขต
│    └──────────────────────────── ปีที่สร้างแฟ้ม
└───────────────────────────────── ประเภทงบ (C = งบ C, I = งบ I)
```

**กฎ node:**
- มีเฉพาะ WBS ที่ขึ้นต้นด้วย `C` เท่านั้น — WBS ที่ขึ้นต้นด้วย `I` ไม่มี node
- node คือ 2 segment สุดท้ายของ WBS แบบ C เช่น `02.2`, `01.1`, `03.1`
- WBS หลัก = ตัดทิ้ง 2 segment สุดท้าย เช่น `C-68-E-KCECS.0080.02.2` → หลัก = `C-68-E-KCECS.0080`

```javascript
function parseWBS(wbs) {
    if (!wbs.startsWith('C-')) {
        return { base: wbs, node: '', budgetOf: '', ownership: '' };
    }
    const parts = wbs.split('.');
    // parts สุดท้าย 2 ตัวคือ node
    const ownership = parts[parts.length - 1];       // '2' หรือ '1'
    const budgetOf  = parts[parts.length - 2];       // '02', '01', '03'
    const node      = `${budgetOf}.${ownership}`;    // '02.2'
    const base      = parts.slice(0, -2).join('.');  // 'C-68-E-KCECS.0080'
    const ownerLabel = ownership === '1' ? 'กฟภ.' : 'ผู้ใช้ไฟ';
    const budgetLabel = budgetOf === '01' ? 'งบของ กฟภ.' :
                        budgetOf === '02' ? 'งบของ ผู้ใช้ไฟ' : 'งบของ อื่นๆ';
    return { base, node, budgetOf, ownership, ownerLabel, budgetLabel };
}
// ตัวอย่าง:
// 'C-68-E-KCECS.0080.02.2' → base='C-68-E-KCECS.0080', node='02.2', budgetOf='02', ownership='2'
// 'I-69-E-KCE69.M4.1104'   → base='I-69-E-KCE69.M4.1104', node='', budgetOf='', ownership=''
```

## ข้อ 1 — เลขรันแฟ้ม + รวมแฟ้มเดียวกัน (สำคัญมาก)

### 1.1 กฎการรวมแฟ้ม
WBS ที่มี base เดียวกัน = แฟ้มงานเดียวกัน ใช้รหัสแฟ้มเดียวกัน:
- `C-68-E-KCECS.0080.01.1` และ `C-68-E-KCECS.0080.02.2` → base = `C-68-E-KCECS.0080` → แฟ้มเดียวกัน
- `I-69-E-KCE69.M4.1104` → base = WBS เต็ม (ไม่มี node) → แฟ้มของตัวเอง

เมื่อ import ZPSR018:
1. parse WBS → หา base
2. เช็ค Google Sheet แท็บ `งบ` ว่า base นี้มีรหัสแฟ้มแล้วหรือยัง
3. มีแล้ว → ใช้รหัสเดิม (เพิ่มข้อมูล node ใหม่เข้าแฟ้มนั้น)
4. ยังไม่มี → รัน KCE01, KCE02... (ใช้ running number ต่อเนื่องจากค่าสูงสุดใน Sheet)

### 1.2 Google Sheet
แท็บ `งบ` เพิ่มคอลัมน์:
- `รหัสแฟ้ม` (KCE01...)
- `WBS_base` (C-68-E-KCECS.0080)
- `node` (02.2 หรือว่างถ้าไม่มี)
- `budgetOf` (01/02/03 หรือว่าง)
- `ownership` (1/2 หรือว่าง)

### 1.3 UI — card แฟ้ม (แนวทาง A: card เดียว แยก tab ตาม node)
Card แสดง:
- หัว: `รหัสแฟ้ม — WBS_base` เช่น `KCE05 — C-68-E-KCECS.0080`
- badge: ประเภทงบ (C/I), ผู้ดำเนินการ (กฟภ./ชื่อบริษัท)
- **tab bar**: ถ้ามีหลาย node → แสดง tab แต่ละ node เช่น `01.1 กฟภ.` `02.2 ผู้ใช้ไฟ`
  - tab ที่เลือกแสดงรายการโครงข่าย + ยอดของ node นั้น
  - ถ้ามี node เดียว → ไม่ต้องแสดง tab bar แสดงรายการโครงข่ายเลย
  - WBS แบบ I ไม่มี node → ไม่แสดง tab bar
- ยอดงบจัดสรรรวม / คงเหลือรวม ของ node ที่เลือก
- progress bar (จ่ายไปแล้ว/งบจัดสรรรวม)

Search box ในหน้า "เลือกแฟ้มงาน": กรองตามรหัสแฟ้ม (KCE01...) หรือ WBS_base
