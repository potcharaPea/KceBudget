# Patch 1 — แก้ไข KceBudget (6 ข้อ)

> วาง prompt นี้เข้า Claude Code ในโฟลเดอร์ `แอปตัดงบก่อสร้าง`
> ทำทีละข้อ แต่ละข้อบอกเมื่อเสร็จแล้วหยุดรอตรวจ

---

## ข้อมูลพื้นฐานที่ต้องรู้ก่อนแก้

**เทมเพลต PDF:** `ใบตัดงบ_template_v9.docx` ใช้ short token เป็น placeholder
- checkbox area มี **ไม่มี placeholder** อยู่เลย — ต้องเพิ่มเข้าไปใหม่ในเทมเพลตด้วย python-docx (ดูข้อ 4 และ 6)
- token ที่มีอยู่แล้ว: `{REF}`, `{JOB}`, `{WBS}`, `{DD}`, `{MM}`, `{YY}`, `{USR}`, `{POS}`, `{APV}`, `{APD}`, `{DPT}`, `{CAT}`, `{NET}`, `{ACT}`, `{YB}`, `{YS}`, `{JB}`, `{JS}`, `{KB}`, `{KS}`, `{CB}`, `{CS}`, `{LB}`, `{LS}`

**ยอดจัดสรรรวม:** ในไฟล์ ZPSR018 บรรทัดท้ายรายงานมีข้อความ:
`"งานก่อสร้างดังกล่าวได้รับจัดสรรงบประมาณจำนวณ [ตัวเลข] บาท"`
ตัวเลขงบจัดสรรคือตัวเลขแรกที่อยู่ในบรรทัดนั้น (เช่น `219,400.00`)
วิธีดึง: หา word ที่ match `\d{1,3}(,\d{3})*\.\d{2}` ที่อยู่ใน y-range ≈ ท้ายหน้า (y > 70% ของ page height) และอยู่ก่อนตัวเลขคงเหลือ — ดึงตัวเลขแรกในบรรทัดนั้น

---

## ข้อ 1 — ยอดจัดสรรรวม

**ปัญหา:** ตอนนี้ยอดจัดสรรรวม (งบทั้ง WBS) ไม่ได้ดึงหรือแสดงในแอป

**แก้:**
1. **parser** — เพิ่ม logic ดึงยอดจัดสรรรวมจากบรรทัดท้ายของ ZPSR018 (ดูข้อมูลพื้นฐานด้านบน) บันทึกลง Google Sheet แท็บ `งบ` คอลัมน์ `ยอดจัดสรรรวม` ของ WBS นั้น
2. **แสดงในแอป** — card ของแต่ละ WBS แสดงยอดจัดสรรรวมด้วย
3. **แก้ไขได้** — เพิ่มปุ่ม edit บน card WBS ให้แก้ยอดจัดสรรรวมได้มือ (กรณีดึงไม่ได้หรือต้องการแก้ไข) พร้อมบันทึกลง Google Sheet
4. **fallback** — ถ้าไฟล์ ZPSR018 ไม่มีบรรทัดยอดจัดสรรรวม (ดึงไม่ได้) ให้แจ้งและเปิดช่องกรอกมือ

---

## ข้อ 2 — นำเข้าไฟล์ ZPSR018 หลายใบพร้อมกัน

**ปัญหา:** ตอนนี้เลือกได้ทีละ 1 ไฟล์เท่านั้น

**แก้:**
- เปลี่ยน `<input type="file">` ให้มี attribute `multiple` → เลือกหลายไฟล์พร้อมกันได้
- วน loop ประมวลผลทีละไฟล์ตามลำดับ แสดง progress (เช่น "กำลังประมวลผล 2/3...")
- สรุปผลเมื่อเสร็จทุกไฟล์: import สำเร็จ X ไฟล์, ข้าม Y ไฟล์ (ยอดเท่าเดิม), พบการเปลี่ยนแปลง Z ไฟล์
- Logic re-import เดิม (เทียบยอด / เตือนถ้าต่าง) ยังคงใช้ต่อทุกไฟล์

---

## ข้อ 3 — Dropdown ผู้เบิก

**ปัญหา:** ตอนนี้ยังไม่มี dropdown ผู้เบิก (หรือมีแต่ไม่สามารถเพิ่มชื่อได้)

**แก้:**
- เพิ่ม dropdown "ผู้เบิก" ในฟอร์มเปิดใบตัดงบ **รูปแบบเดียวกับ dropdown พขร.**
- ดึงรายชื่อจาก Google Sheet แท็บ `ตั้งค่า` (เพิ่มคอลัมน์ `ผู้เบิก` ถ้ายังไม่มี)
- มีปุ่ม "เพิ่มชื่อใหม่" inline — กรอกชื่อแล้วบันทึกขึ้น Sheet ทันที dropdown อัปเดตโดยไม่ต้อง reload

---

## ข้อ 4 — PDF ไม่แสดงข้อความในช่อง checkbox หน้างาน

**ปัญหา:** ช่องต่อไปนี้มีให้กรอกในแอป แต่ไม่ปรากฏในไฟล์ PDF เพราะเทมเพลตไม่มี placeholder:
- ค่าแรง พขร.(บ.) — (เครนสว่านเจาะ) → ชื่อ พขร. และเครนสว่านเจาะ
- ค่าแรง พขร.(บ.) — (รถกระเช้า) → ชื่อ พขร. และรถกระเช้า
- ค่าแรงคนงานรายวัน งวด วันที่ .... ทีม ....
- ค่าแรงจ้างเหมา สัญญาจ้างที่ ... ลว. ...

**แก้ — เพิ่ม placeholder ในเทมเพลต:**

แก้ไฟล์ `ใบตัดงบ_template_v9.docx` โดยใช้ python-docx เพิ่ม token ในแต่ละ paragraph ของ checkbox area (อยู่ใน c0 ของ row 4):

| Paragraph เดิม | Token ที่ต้องแทรก | ตำแหน่ง |
|---|---|---|
| `(    )  ค่าแรง พขร.(บ.) ......... (เครนสว่านเจาะ)` | `{CRN_NAME}` | แทนที่ `..........` |
| `(    )  ค่าแรง พขร.(บ.) ......... (รถกระเช้า)` | `{CRT_NAME}` | แทนที่ `..........` |
| `(    )  ค่าแรงคนงานรายวัน งวด วันที่ ... ทีม ...` | `{DLY_DATE}` และ `{DLY_TEAM}` | แทนที่ `...` แต่ละส่วน |
| `(    )  ค่าแรงจ้างเหมา สัญญาจ้างที่ ...` | `{CON_NO}` | แทนที่ `...` |
| `                                ลว. ...` | `{CON_DATE}` | แทนที่ `...` (บรรทัดลว.) |

บันทึกเป็น `ใบตัดงบ_template_v10.docx`

## ข้อ 5 — แยกช่องสัญญาจ้างที่ และ ลว.

**ปัญหา:** ตอนนี้กรอกรวมกันเป็นช่องเดียว

**แก้:**
- ฟอร์มเปิดใบตัดงบ: แยก input เป็น 2 ช่อง — "สัญญาจ้างที่" และ "ลว."
- ส่งค่าแยกกันให้ GAS แล้ว replaceText `{CON_NO}` และ `{CON_DATE}` ตามลำดับ
- ถ้า checkbox ค่าแรงจ้างเหมาไม่ถูกติ๊ก ให้ใส่ `''` (ว่าง) ทั้งสองช่อง

---

## ข้อ 6 — ติ๊กถูกใน PDF ตามที่เลือกในแอป

**ปัญหา:** ช่อง `(    )` ในเทมเพลตเป็น text ตายตัว ไม่มี placeholder ทำให้ PDF แสดงเป็นว่างเสมอ

**แก้ — เพิ่ม placeholder สำหรับ checkbox ทุกตัว:**

แก้ไฟล์ `ใบตัดงบ_template_v9.docx` เพิ่ม token แทน `(    )` ดังนี้:

| Paragraph | Token checkbox | ค่าถ้าติ๊ก | ค่าถ้าไม่ติ๊ก |
|---|---|---|---|
| ค่ายานพาหนะ/เบี้ยเลี้ยง พชง | `{CHK_VEH}` | `(✓)` | `(    )` |
| ค่าพาหนะ/เบี้ยเลี้ยง พชร(บ) | `{CHK_TRV}` | `(✓)` | `(    )` |
| ค่าแรง พขร.(บ.) เครนสว่านเจาะ | `{CHK_CRN}` | `(✓)` | `(    )` |
| ค่าแรง พขร.(บ.) รถกระเช้า | `{CHK_CRT}` | `(✓)` | `(    )` |
| ค่าแรงคนงานรายวัน | `{CHK_DLY}` | `(✓)` | `(    )` |
| ค่าแรงจ้างเหมา | `{CHK_CON}` | `(✓)` | `(    )` |
| ค่าน้ำมันยานพาหนะ | `{CHK_OIL}` | `(✓)` | `(    )` |
| อื่นๆ | `{CHK_OTH}` | `(✓)` | `(    )` |

ใน GAS: `body.replaceText('{CHK_VEH}', isChecked ? '(✓)' : '(    )')` ทุกตัว

**รวมข้อ 4+6:** แก้เทมเพลตครั้งเดียว บันทึกเป็น `ใบตัดงบ_template_v10.docx` แล้วอัปโหลดขึ้น Google Drive แทนตัวเก่า
